## Description

Server for skk system. It is made in Nestjs.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

##### Run all migrations before starting the project

```bash
npm run typeorm:run-migrations
```

## Configurations

- DB_PORT=5432
